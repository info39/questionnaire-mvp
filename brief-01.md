\# Desert Pulse — Generative Art Brief



\## Concept

Create a generative art concept called "Desert Pulse".



The artwork should express:

\- desert winds

\- silence and vast space

\- organic motion

\- tension between randomness and control



\## Style Direction

\- Minimal and premium

\- Subtle and elegant (not loud or flashy)

\- Avoid cyberpunk / neon styles

\- Focus on calm, refined motion



\## Technical Requirements

\- Use p5.js for the artwork

\- Include seeded randomness (same seed = same output)

\- Smooth motion, not chaotic



\## Deliverables

1\. A philosophy markdown file explaining the concept

2\. A JavaScript file with the generative art (p5.js)

3\. An HTML file to display the artwork



\## Output Instructions

\- Save all files inside the `/output` folder

\- Do not only explain — actually create the files
